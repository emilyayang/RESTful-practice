# RESTful-practice

A RESTful service that retrieves a string stored in a database and displays it on the page 
# RESTful-practice

## Install

    $ git clone https://github.com/emilyayang/RESTful-practice.git
    $ cd RESTful-practice
    $ npm install

## Start & Watch

    $ npm start

## Create Database
Open mysql shell

    mysql> create database names;

## Seed Database

    $ npm run seed

## Build for production

    $ npm run build
